# -resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO
- Criação de conta azure
- difereça entre capEx e Opex 

CapEx (Capital Expenditure)
É investimento em bens físicos — você compra e passa a ser dono.
Exemplo tradicional (on‑premise):

Comprar servidores
Comprar storage
Montar datacenter

Características:

 Alto custo inicial
 Você é dono do equipamento
 Manutenção por sua conta
 Deprecia com o tempo


 OpEx (Operational Expenditure)
É gasto operacional recorrente — você paga pelo uso.
No Azure:

VMs (máquinas virtuais)
Storage (Blob, Disk)
Banco de dados (SQL, Cosmos)
Serviços como App Service

Características:

 Pagamento conforme uso (pay-as-you-go)
 Sem investimento inicial alto
 Escalável (sobe/desce conforme necessidade)
🔧 Manutenção fica com a Microsoft
